Este proyecto es una Api-RestFull que tiene como fin crear, actualizar, eliminar, ver los cursos que contiene la base de datos, para una comprencion mas facil este proyecto utiliza la estructura MVC pero en lugar de la carpeta vistas esta la carpeta rutas. 

Ademas de manipular la informacion de los cursos, tambien esta api tiene la funcion de crear y verificar usuarios por medio de contraseñas creadas automaticamente en base64 para que solo pueda manipular la informacion correspondiente a este usuario, por ejemplo, si se quiere eliminar o actualizar un curso se pedira su token y contraseña o clave para poder efectuar dicha funcion.
